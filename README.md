# Capgemini Data Engineering Training

This repository contains hands-on exercises, assignments, and learning modules from the Capgemini Data Engineering Training program. It is structured to guide learners from foundational data engineering concepts to practical business analytics using SQL and PySpark.

## 📚 Repository Structure

- **Week-0/**: Core learning pathway, divided into four phases:
	- **Phase-0/**: *Associate Data Engineering* foundational modules (Lakeflow, Spark, DevOps, Data Ingestion)
	- **Phase-1/**: SQL to PySpark basics, DataFrame operations, and query translation
	- **Phase-2/**: Customer sales analysis using PySpark, advanced joins, aggregations, and business insights
	- **Phase-3/**: (Reserved for future content)....
- **mini-project/**: (Reserved for capstone or additional projects)
- **Week-1/**, **Week-2/**, **Week-3/**: (Reserved for further modules or assignments)

## 📝 Key Learning Outcomes

- Data ingestion and pipeline building with Lakeflow and Spark
- Workflow automation, job scheduling, and DevOps basics
- SQL and PySpark DataFrame operations: filtering, selecting, grouping, and aggregating data
- Translating SQL queries to PySpark code
- Real-world business analytics: customer sales analysis, revenue computation, and data transformation

## 🗂️ Notable Files & Folders

- `Week-0/Phase-0/`: Contains module PDFs and a summary README of foundational data engineering concepts
- `Week-0/Phase-1/pySpark.py`: PySpark basics and DataFrame operations
- `Week-0/Phase-1/sql_queries.sql`: Equivalent SQL queries for comparison
- `Week-0/Phase-2/assignment_customer_sales_analysis.py`: Customer sales analytics using PySpark
- `outputs/`: Contains output images from code execution for reference

## 🚀 Getting Started

1. Clone this repository
2. Review the README in each phase for detailed learning notes
3. Explore the code and outputs to understand practical data engineering workflows
