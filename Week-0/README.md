### **Phase 1 – SQL to PySpark**

* Learned PySpark DataFrame basics (`select`, `filter`, `groupBy`)

* Understood DataFrame as structured data

### **Phase 2 – SQL Joins & Aggregations**

* Mastered joins (`INNER`, `LEFT`, `RIGHT`, `FULL`)
* Applied `GROUP BY`, `COUNT`, `SUM`, `AVG`
* Solved real-world SQL problems

✅ **Outcome:** Strong understanding of relational data analysis

### **Phase 3 – SQL Conditional Logic & Window Functions**

* Applied `CASE WHEN` for business logic
* Used window functions (`ROW_NUMBER`, `RANK`, `DENSE_RANK`)
* Performed partition-based analysis

✅ **Outcome:** Ability to solve advanced analytical SQL problems

---

### **Phase 4 – Databricks & Data Cleaning**

* Worked with Databricks (clusters, notebooks, DBFS)
* Performed data cleaning (`dropna`, `fillna`, `dropDuplicates`)

✅ **Outcome:** Hands-on experience with Databricks environment

### **Phase 5 – End-to-End Pipeline (Olist Dataset)**
